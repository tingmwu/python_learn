Arch安装后的配置（超详细）

1. 添加archlinuxcn源#

在 /etc/pacman.conf 文件末尾添加以下两行： 

Copy
[archlinuxcn]
Server = https://mirrors.tuna.tsinghua.edu.cn/archlinuxcn/$arch

然后安装 GPG key

Copy
sudo pacman -Syu
sudo pacman -S archlinuxcn-keyring

2. 配置aur#

安装yay

Copy
sudo pacman -S yay

修改aururl 

Copy
yay --aururl "https://aur.tuna.tsinghua.edu.cn" --save

3. 更换shell为zsh# 

Copy
chsh -s /bin/zsh

重启即可生效

4. 安装oh-my-zsh及插件#

先确保git，wget, curl已经安装

Copy
sudo pacman -S git wget curl

安装ohmyzsh

Copy
sh -c "$(wget https://raw.github.com/ohmyzsh/ohmyzsh/master/tools/install.sh -O -)"

安装历史记录插件和语法检查插件

Copy
cd ~/.oh-my-zsh/plugins
git clone git://github.com/zsh-users/zsh-autosuggestions.git
git clone https://github.com/zsh-users/zsh-syntax-highlighting.git

下载好后在～/.zshrc文件中加入上述插件

找到plugins=(git)，改为如下（sudo插件无须下载，效果为连按两次esc键给命令加上sudo） 

Copy
plugins=(
	git
	sudo
	zsh-syntax-highlighting
	zsh-autosuggestions
)

使插件生效

Copy
source ~/.zshrc

5. 安装输入法#

安装fcitx

Copy
sudo pacman -S fcitx-im
sudo pacman -S fcitx-cofigtool

在家目录下创建.xprofile文件并写入以下内容 

Copy
export XIM=fcitx
export XIM_PROGRAM=fcitx
export GTK_IM_MODULE=fcitx
export QT_IM_MODULE=fcitx
export XMODIFIERS="@im=fcitx"

然后安装喜欢的输入法，这里推荐实用讯飞输入法（搜狗输入法有bug可能用不了）

讯飞输入法需要去aur里安装

Copy
yay -S iflyime

重启查看fcitx 配置是否已经添加了讯飞输入法

6. 配置开发环境#

python#

下载pip

Copy
sudo pacman -S python-pip

设置pip源

Copy
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple

golang#

  1. 安装go

    Copy
    sudo pacman -S go

  2. 选择一个go工作目录

    我这里以～/Documents/go为例

    在～/Documents/go下新建三个文件夹分别为src,bin,pkg

  3. 配置环境变量（注意改为你自己的工作目录,GOROOT=/usr/lib/go和我一样，这是安装目录）

    编辑～/.xprofile 

    Copy
    export GOROOT=/usr/lib/go
export GOPATH=~/Documents/go		# 这两行你需要 
export GOBIN=~/Documents/go/bin		# 修改为自己的
export PATH=$PATH:$GOROOT/bin:$GOBIN

    然后

    Copy
    source .xprofile

  4. 配置GOPROXY

    Copy
    go env -w GOPROXY=https://goproxy.io,direct

java#

查看当前系统中已经存在的jdk有哪些

Copy
archlinux-java status

选中其中一个为默认jdk

Copy
sudo archlinux-java set `上面列表中列出的名字其中一个`

安装最新版jdk

Copy
sudo pacman -S jdk

安装jdk8

Copy
sudo pacman -S jdk8-openjdk

nodejs#

安装

Copy
sudo pacman -S nodejs npm

npm使用淘宝镜像

Copy
npm config set registry https://registry.npm.taobao.org

安装vue-cli

Copy
npm install -g @vue/cli

docker#

安装docker

Copy
sudo pacman -Syu docker

免sudo执行docker

Copy
sudo gpasswd -a ${USER} docker

配置docker国内镜像

在/etc/docker目录下新建daemon.json文件,写入下面内容(网易镜像) 

Copy
{
	"registry-mirrors": ["http://hub-mirror.c.163.com"]
}

重启docker生效

Copy
sudo systemctl restart docker

mysql#

  1. 安装Maria DB
Copy
sudo pacman -S mariadb
  1. 配置目录
Copy
sudo mariadb-install-db --user=mysql --basedir=/usr --datadir=/var/lib/mysql
  1. 启动Maria DB
Copy
sudo systemctl start mysqld
  1. 为root用户设置一个新密码
Copy
sudo mysqladmin -u root password '123456'

至此已经可以进入数据库了，命令mysql -u root -p

  1. root用用户开启远程访问

    先进入数据库

    1. 选中mysql数据库
    Copy
    use mysql;

  2. 更改root的访问权限

    可以先查看当前权限信息（不是必须）

    select user,authentication_string,host from user;

    123456 为你设置的root用户密码

    Copy
     GRANT ALL PRIVILEGES ON *.* TO 'root'@'%' IDENTIFIED BY '123456';
    1. 刷新数据
    Copy
    flush privileges;

  3. 将mysql设为开机自启

Copy
  sudo systmctl enable mysql

8. 美化grub启动界面#

  1. 去商店下载主题包

    gnome-look

    kde-look

    访问有点慢。。。

  2. 解压下载好的主题

    Copy
    sudo tar -xf 主题包名

  3. 复制到grub主题目录

    Copy
    sudo cp -r 主题包名 /usr/share/grub/themes/

  4. 修改文件添加主题

    Copy
    sudo vim /etc/default/grub

    找到#GRUB_THEME=去掉注释，该为对应的主题名称，就像这样 

    Copy
    GRUB_THEME="/usr/share/grub/themes/主题包名/theme.txt"
